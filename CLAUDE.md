# Quack Studio

Native mobile mini-game collection — pivoted from the web prototype at
`../quack-blaster/` (hub, duck mascot, currencies, daily quests, 2 mini-games)
per an external studio-brief ("master prompt") calling for a premium 5-game
collection with collectible mascots, IAP, and live-ops.

## Technology Stack

- **Engine**: Unity 6.3 LTS
- **Language**: C#
- **Build System**: Unity Build Pipeline
- **Asset Pipeline**: Unity Asset Import Pipeline + Addressables
